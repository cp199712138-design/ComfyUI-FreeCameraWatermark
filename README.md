# ComfyUI-FreeCameraWatermark

轻量相机水印节点。复制到 ComfyUI 的 `custom_nodes` 后重启即可使用，不需要 pip，不需要 comfy-env。

## 安装

下载或解压本仓库，把整个文件夹放到：

```text
ComfyUI/custom_nodes/ComfyUI-FreeCameraWatermark
```

重启 ComfyUI，搜索：

```text
自由相机水印
```

也可以搜索：

```text
Free Camera Watermark
```

## 怎么用

1. 主图连接到 `image`。
2. 如果要用签名或 Logo，把透明图接到 `logo`，遮罩接到 `logo_mask`。
3. 选择模式：`文字`、`Logo`、`Logo+文字`、`相机白条`、`透明水印`、`图案水印`。
4. 用 `主颜色` 的颜色按钮选择颜色，用 `透明度` 滑块控制浓淡。
5. 在节点下方参考画布里拖动蓝框移动位置，拉右下角手柄调整大小。
6. 节点执行后，参考画布会优先显示当前预览图，方便看着图调位置。

## 模式说明

- `文字`：只叠加文字。
- `Logo`：叠加签名或 Logo，保留原图颜色，只用透明度控制浓淡。
- `Logo+文字`：Logo 和文字放在同一个框里。
- `相机白条`：生成底部相机参数白条，白条高度由拖拽框高度控制。
- `透明水印`：用低透明度文字或 Logo 做水印。
- `图案水印`：生成轻量装饰图案；默认 `渐变光影`，也可选 `科幻光栅`，不联网下载任何图片。

## 操作提示

- 点参考画布任意位置，水印框会移动到那里。
- 拖蓝框内部可以继续移动。
- 拖右下角蓝色手柄可以缩放。
- 缩小后也不用精准点边框，命中范围已经放大。
- 底部会显示位置、宽高百分比和大约像素尺寸。
- `透明度`：0 是看不见，100 是不透明。
- `主颜色` 使用浏览器原生颜色选择器，旁边保留十六进制颜色输入，懂颜色值的人也可以直接输入。

## 说明

这个节点只把水印合成到图片像素里，不上传文件，不清理图片元数据。需要无元数据保存时，可以搭配 `Save Image Without Metadata` 使用。

# Optional Fonts

This folder is optional.

The node already works with bundled Latin fonts and common system fonts. If you want prettier Chinese styles, place font files here with these names:

```text
LXGWWenKai-Regular.ttf
SmileySans-Oblique.ttf
```

Then restart ComfyUI and choose:

- `CJK Handwritten Optional`
- `CJK Display Optional`

Do not put model files or generated images in this folder.
